# 🚗 Trip Planner - Sua Viagem sob Controle

O **Trip Planner** é um aplicativo Android moderno desenvolvido para ajudar viajantes a planejar o custo de suas viagens de forma simples, rápida e intuitiva. Com uma interface premium e focada em UI/UX, o app guia o usuário através de um fluxo otimizado para calcular gastos com combustível.

## ✨ Funcionalidades

*   **Splash Screen Animada:** Início fluido com animação de rotação e transição suave.
*   **Tela Inicial Dinâmica:** Apresentação dos pilares do app (Planeje, Viaje, Economize) com troca automática de ícones e textos.
*   **Fluxo de Input Inteligente:**
    *   Filtros que impedem a digitação de letras ou caracteres especiais.
    *   Conversão automática de vírgula para ponto decimal.
    *   Validação de campos para evitar cálculos vazios.
*   **Cálculo Preciso:** Fórmula baseada em distância, consumo médio e preço do combustível.
*   **Resumo Estilo "Trip Ticket":** Visualização elegante dos dados inseridos e do custo total estimado.
*   **Bottom Sheets & Dialogs:** Informações extras e confirmações de segurança de forma contextual.

## 🛠️ Tecnologias Utilizadas

*   **Kotlin** - Linguagem oficial para desenvolvimento Android.
*   **Jetpack Compose** - Toolkit moderno para construção de interfaces nativas.
*   **Material Design 3** - Cores e componentes seguindo as últimas diretrizes de design.
*   **Compose Navigation** - Navegação fluida entre telas.
*   **ViewModel** - Gerenciamento de estado e lógica de negócio.
*   **Vector Assets** - Ícones e ilustrações customizadas e escaláveis.

## 🎨 Identidade Visual

O projeto utiliza uma paleta de cores personalizada baseada em tons de **Roxo (Purple)** e **Turquesa (Teal)**, garantindo um visual vibrante e profissional.

## 🚀 Como Executar o Projeto

1.  Clone este repositório:

3.  Aguarde a sincronização do Gradle.
4.  Execute no seu emulador ou dispositivo físico.
